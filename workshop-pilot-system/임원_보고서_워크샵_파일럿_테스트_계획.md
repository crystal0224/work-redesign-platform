# 워크샵 재설계 파일럿 테스트 실행 계획서

**작성일**: 2025년 11월 23일
**보고 대상**: 임원진
**작성자**: Work Redesign Platform 개발팀

---

## 📋 Executive Summary

Work Redesign Platform 워크샵의 실전 배포 전, **AI 기반 시뮬레이션을 통한 30명 팀장 대상 파일럿 테스트**를 실시하여 워크샵 효과성을 사전 검증하고 개선점을 도출하고자 합니다.

### 핵심 목표
- ✅ 다양한 직무/디지털 성숙도의 30명 팀장 페르소나로 실전 시나리오 시뮬레이션
- ✅ 11단계 워크샵 각 단계별 만족도, 이탈 위험, 소요 시간 측정
- ✅ 직무별/수준별 개선 필요 사항 구체적 파악
- ✅ 실제 HRD 운영 시 예상되는 리스크 사전 식별 및 대응 방안 수립

### 기대 효과
- **품질**: 실제 교육 전 시뮬레이션으로 워크샵 완성도 대폭 향상
- **리스크 관리**: 이탈 위험 높은 그룹 사전 파악 및 맞춤형 지원 계획 수립
- **비용 절감**: 실제 30명 파일럿 대비 **시간 90% 절감** (1-2시간 내 완료)
- **데이터 기반 의사결정**: 객관적 데이터로 워크샵 개선 우선순위 결정

## 3️⃣ 파일럿 테스트 실행 방법론

### 3.1 AI 기반 시뮬레이션 시스템

**기술 스택**
- **AI 모델**: Claude 3.5 Sonnet (Anthropic)
- **시뮬레이션 엔진**: TypeScript 기반 맞춤형 개발
- **페르소나 데이터**: 30명 상세 프로필 (직무, 팀 구성, Pain Points, 우려사항 등)
- **워크샵 UI 스펙**: 11단계 전체 화면/기능 명세

### 3.2 3단계 피드백 수집

#### Phase 1: 사전 인터뷰 (Pre-Interview)
- **목적**: 각 페르소나의 초기 기대/우려 파악
- **질문**:
  - 워크샵 참여 동기는?
  - 사전 우려사항은?
  - 3시간이 적절한가?
- **소요 시간**: 페르소나당 약 2분

#### Phase 2: 워크샵 실행 + 중간 체크인
- **목적**: 11단계 워크샵 실제 진행 시뮬레이션
- **측정 항목**:
  - 각 단계별 소요 시간
  - 사용 편의성 (1-10점)
  - 명확성 (1-10점)
  - 가치 인식 (1-10점)
  - Pain Points (구체적 서술)
  - Positive Points (구체적 서술)
  - 계속 진행 의향 (Yes/No)
- **중간 체크인**: 각 단계 완료 후 즉시 피드백 수집
- **소요 시간**: 페르소나당 약 45-60분

#### Phase 3: 사후 인터뷰 (Post-Interview)
- **목적**: 전체 워크샵 경험에 대한 종합 평가
- **질문**:
  - 전반적 만족도 (1-10점)
  - 추천 의향 (Yes/No)
  - 최우선 개선 사항 3가지
  - 실제 팀 적용 계획
- **소요 시간**: 페르소나당 약 3-5분

### 3.3 병렬 실행 전략 (시간 최적화)

**배치 구성**
- 총 30명을 **6개 그룹**으로 나눔 (각 5명)
- 6개 그룹을 **2개 배치**로 실행 (각 배치 3개 그룹 병렬)

**실행 계획**
```
📦 배치 1 (15명 병렬) → 60초 대기 → 📦 배치 2 (15명 병렬)
```

| 배치 | 그룹 | 직무 | 인원 |
|---|---|---|---|
| **배치 1** | Group 1 | Marketing & Sales | 5명 |
| | Group 2 | Production & Operations | 5명 |
| | Group 3 | R&D & Innovation | 5명 |
| **60초 대기** | (Rate limit 해소) | | |
| **배치 2** | Group 4 | HR & Finance | 5명 |
| | Group 5 | IT & Digital | 5명 |
| | Group 6 | Strategy & Planning | 5명 |

**예상 소요 시간**
- 배치 1 실행: ~30분
- 배치 간 대기: 1분
- 배치 2 실행: ~30분
- 분석 및 보고서 생성: ~10분
- **총 소요 시간**: **60-70분**

💡 **시간 절감 효과**: 순차 실행 시 450분(7.5시간) → 병렬 실행 60분 = **87% 시간 절감**

## 4️⃣ 측정 지표 및 성공 기준

### 4.1 핵심 측정 지표

| 지표 | 목표 | 측정 방법 |
|---|---|---|
| **완료율** | 85% 이상 | 30명 중 11단계 완료 비율 |
| **평균 만족도** | 7.0/10 이상 | 사후 인터뷰 전반적 만족도 |
| **추천 의향** | 75% 이상 | "타 팀장에게 추천하겠습니까?" Yes 비율 |
| **이탈 위험** | 5명 이하 | 중도 포기 또는 매우 낮은 만족도 |
| **소요 시간** | 평균 180분±30분 | 워크샵 전체 소요 시간 |

### 4.2 단계별 상세 분석

각 11단계별로 다음 측정:
- 평균 소요 시간
- 평균 만족도 (ease, clarity, value)
- 주요 Pain Points TOP 3
- 주요 Positive Points TOP 3
- 시간 인식 (Too Short / Just Right / Too Long)

### 4.3 직무별/수준별 세분화 분석

**직무별 비교**
- Marketing & Sales vs Production vs R&D vs Staff
- 각 직무별 만족도, 이탈률, 개선 요청 사항
