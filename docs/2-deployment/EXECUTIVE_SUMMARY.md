# Work Redesign Platform - 임원 보고 자료

> **목적**: 2025년 3월 팀장 교육과정에서 활용할 업무 자동화 워크샵 플랫폼 구축

---

## 📌 한 줄 요약

**50명 이상의 팀장이 동시에 AI 기반 업무 자동화 워크샵을 진행할 수 있는 클라우드 플랫폼을 구축하여, 3월 교육과정부터 활용**

---

## 🎯 사업 목적

### 현재 문제점
- 팀장들의 업무 자동화 니즈는 높으나, 기술적 장벽으로 실행 어려움
- 수작업으로 워크샵 진행 시 시간 소요 과다 (1회당 8시간+)
- 결과물 품질 편차 크고, 지속적 관리 어려움

### 솔루션
- **AI 기반 자동화 분석**: Claude API로 업무 자동화 가능성 자동 분석
- **대화형 컨설팅**: AI와 대화하며 맞춤형 자동화 솔루션 설계
- **즉시 활용 가능한 결과물**: Python/n8n 워크플로우 자동 생성

### 기대 효과
- 워크샵 시간 **70% 단축** (8시간 → 2.5시간)
- 팀장 1명당 연간 **240시간 업무 시간 절감** (자동화 적용 시)
- 전사 업무 효율화 문화 확산

## 🎓 3월 교육과정 계획

### 교육 대상
- SK 각 부문 팀장급 50명 이상
- 향후 확대 가능 (분기별 100명+)

### 교육 내용 (1회 2.5시간)
1. **업무 현황 분석** (30분)
   - 팀 업무 영역 정의
   - 현재 업무 리스트 입력/업로드

2. **AI 자동화 분석** (40분)
   - AI가 자동으로 업무 분석
   - 자동화 가능성 평가
   - 우선순위 추천

3. **솔루션 설계** (60분)
   - AI 컨설턴트와 대화형 설계
   - 워크플로우 자동 생성
   - 구현 가이드 제공

4. **결과물 다운로드** (20분)
   - Python 스크립트
   - n8n 워크플로우
   - 실행 매뉴얼

### 결과물
- 팀별 맞춤형 자동화 솔루션
- 즉시 실행 가능한 코드
- 구현 가이드 및 ROI 분석
